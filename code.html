Fishlord Worship

Click to Run Code

Fishlord Worship Simulator is a small program made using p5 javascript. This game was created so I could familiarize myself with coding games and their unique user interfaces. The gameplay is incredibly simple. Different kinds of fish can be found in the pond, by grabbing them and presenting them to the Fishlord, you will receive different kinds of advice.

The concept for the Fishlord Worship Simulator came from my fascination with dark fantasy and eldritch monsters mixed with the inherent goofy appearance of fish. Those points of inspiration led me to create the Fishlord, who is presented as a mysterious monster that dwells in an underground temple. The dark atmosphere contrasts the silly looking fish and absurd advice.

This drawing program utilizes the different unique shapes to create interesting geometric designs. The 4 different brushes are meant to play on the unique ways shapes can be used to create patterns and visuals.
